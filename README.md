## Requirements
1. `conda`
2. `jupyter-notebook`
3. `nltk`

## Instructions to Run
```
$ git clone https://github.com/muskanrastogi1/NLP-Tasks
$ cd NLP-Tasks
$ jupyter-notebook
```
